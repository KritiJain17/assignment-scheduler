About the Assignment Section


Assignment Section in teams is a simple way of assigning and submitting projects.This project has different modules.


Account Page
  • The signin section will have the facility for both students and teachers to login into their respective accounts.
  • The signup section will have the facility for both students and teachers to signup for their respective accounts.>

    • After signing in their accounts

    Student Page
    The Student's Page will have two subcategories
    • The Pending assignment has the Code of assignment, Link to the assignment and a form to submit their work.
    • The Complete Assignments has the Code of assignment, Link to the assignment and the work submitted by student.

    Faculty Page
    The Faculty's Page will have two subcategories as well
    • Given Assignment has Name of the Assignment, Link to the Assignment, Code of the assignment and a form to get the details of the students who have submitted that assignment.
    • Give a New Assignment allows the faculty to schedule a new assignment for the students. It takes Assignment Name, Assignment Link, Faculty ID, Assignment Code and schedules it for the students.

    Contact Page
    There is a Contact page as well which allows you to communicate to the owner of the page.It asks the user to enter their Name, Email ID and Message.
    We hope that you have a great time using this application.
